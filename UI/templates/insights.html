Insights about decision making

The basis of this algorithm is a Million Musical Tweets, a survey that collected what songs users mentioned to
listen to on microblogs and social media. For each sample the geo location and a date is provided.

Here's an overview from where data possibly could be extracted from.


Since API requests are limited, the original application only considered data from Europe.
As the variance of meterological and cultural differences is rather high, this can be considered
as a meaningful subset.
